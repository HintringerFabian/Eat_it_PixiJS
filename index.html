PixiJS

Was ist PixiJS?

PixiJs ist eine 2D Web Creation Engine, welche WebGL benutzt, um visuellen Kontent darzustellen. Im Gegensatz zu Flash stellt PixiJs eine bessere Performance und gute Effekte auf Pixelebene zur Verfügung. Vorallem für Online Spiele, Datenvisualisierung, interaktive Grafiken und Animationen ist PixiJs perfekt geeignet.

Wie funktioniert PixiJS?

  • 1. HTML File erstellen
  • 2. Web Server erstellen
  • 3. Pixi.min.js einbinden
  • 4. Applikation erstellen
  • 5. Applikation View hinzufügen
  • 6. Sprite erstellen

Was zeichnet PixiJs aus?

Open Source

PixiJs ist Open Source und kann kostenlos verwendet werden. Außerdem gibt es zahlreiche Beispiele und Tutorials auf der PixiJs Homepage. PixiJs wird auch auf Github gehostet, um Probleme verfolgen zu können und den Zugriff zu erleichtern.

Geschwindigkeit

Die Render-Pipeline wurde so entwickelt, dass sie die größtmögliche Leistung aus den Browsern ihrer Nutzer herausholt.

Flexibilität

PixiJs ist sehr flexibel und kann für viele verschiedene Zwecke verwendet werden. Es ist möglich, PixiJs mit anderen Frameworks zu kombinieren, um so eine optimale Lösung zu erhalten.

WebGL Native

Die Nutzung von WebGL ermöglicht PixiJs nicht nur eine bessere Performance, sondern man kann Filter-Klassen verwenden, mit denen man Shader-Programme schreiben bzw verwenden kann. Dadurch lassen sich viele verschiedene visuelle Effekte erzielen.

CodeBeispiel

Dies ist nur ein Beispiel für die Abgabe, es hat nichts mit unserer Website zu tun

                
                    const app = new PIXI.Application({ width: 800, height: 600, backgroundColor: 0x1099bb });
                    document.body.appendChild(app.view);

                    const bunny = PIXI.Sprite.from('assets/basics/bunny.png');

                    bunny.anchor.set(0.5);

                    bunny.x = app.screen.width / 2;
                    bunny.y = app.screen.height / 2;

                    app.stage.addChild(bunny);

                    app.ticker.add((delta) => {
                        bunny.rotation += 0.01 * delta;
                    });

Zusammenfassung

Herausforderungen und Erfahrungen

Anfangs mussten wir uns mit den Grundlagen von PixiJs vertraut machen, da keiner von uns schon vorher damit gearbeitet hatte. Die Homepage von PixiJs bietet hier reichlich Möglichkeiten sich einzulesen und Beispiele zu einzelnen Funktionen zu finden.

Schnell haben wir gemerkt, dass es zahlreiche Möglichkeiten gibt und dass wir uns auf ein paar wenige Funktionen beschränken mussten, um eine funktionierende Applikation zu erstellen, die nicht den Rahmen sprengen würde. Ein Problem, auf das wir gestoßen sind, war die Tatsache, dass wir einen WebServer brauchten, um die erstellten Sprites anzeigen zu können. Diese Hürde war aber schnell genommen. Da es einige Informationen und Hilfestellung zu PixiJs im Internet gibt, war uns immer geholfen.
